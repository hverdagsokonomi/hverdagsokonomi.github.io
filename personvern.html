Personvernerklæring

Denne erklæringen forklarer hvilke data Hverdagsøkonomi kan behandle, og hvordan kalkulatorer, samtykkevalg og kontakt fungerer.

Hvilke data behandles?

Nettsiden har ingen innlogging, database eller backend. Ved vanlig bruk behandles tekniske nettleserdata av GitHub Pages som vertstjeneste. Hverdagsøkonomi legger ikke inn egne sporingsskript uten samtykke.

Kalkulatorer kjører lokalt

Kalkulatorene kjører lokalt i nettleseren. Hverdagsøkonomi samler ikke inn tallene du skriver inn i kalkulatorene, og kalkulatordata sendes ikke til en server.

Kontakt via e-post

Hvis du sender e-post til kontakt@example.com, behandles meldingen manuelt for å kunne svare. Ikke send sensitive personopplysninger, helseopplysninger, fødselsnummer, kontonummer eller detaljerte forsikrings- og lånedokumenter.

Google AdSense og annonser

Hverdagsøkonomi kan bruke Google AdSense som annonsepartner etter gyldig samtykke til annonsering. Det er ikke lagt inn ekte publisher-ID i denne versjonen, og annonse-/tracking-script skal ikke aktiveres før samtykke foreligger.

Samtykkevalg i localStorage

Lokale samtykkevalg kan lagres i nettleserens localStorage. Dette brukes for å huske om du har valgt nødvendige, analyse eller annonsering. Du kan endre valgene på cookiesiden.

Kontaktinformasjon

Kontakt: kontakt@example.com. Dette er en plassholder som bør byttes før publisering.
